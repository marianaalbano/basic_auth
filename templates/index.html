Selecione a opção

Este é um sistema básico para adicionar, fazer a autenticação de usuários e realizar a troca de senha para o primeiro acesso.

Adicionar usuários

  • Para adicionar usuários
  • deve ser realizado o upload
  • de um arquivo do tipo CSV.
  • As informações de usuários dentro do arquivo
  • devem ser separadas por vírgula (,)

Acessar sistema

  • Usuários que fizerem login
  • pela primeira vez devem
  • alterar a sua senha.
  • Utilize sempre uma senha segura contendo:
  • números, letras e caracteres especiais
